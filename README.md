# Installation & Building

1. Install gulp then yarn

2. # Use branch Dev
$ git checkout dev

# you must go into the SRC directory to run this theme.
$ cd simply/src

# install the dependencies in the src directory
yarn

# run build & livereload task
$ yarn dev

# link to ghost themes dir
$ ln -s $PWD path/to/ghost/content/themes/simply
